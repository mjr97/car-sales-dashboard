# Car Sales Dashboard

Interactive dashboard developed to analyze used/new car sales trends, pricing distribution, and market patterns.

## Live Demo
[Open App](https://car-sales-dashboard-z736.onrender.com/)

## Project Objective
Provide an interactive tool for exploring automotive sales data through filters and visual analytics.

## Features
- Interactive filters
- Price distribution histograms
- Mileage vs Price scatter plots
- Brand / model segmentation
- Market trend exploration

## Tech Stack
- Python
- Pandas
- Streamlit
- Plotly / Matplotlib
- Render

## Dataset
Car sales dataset containing vehicle prices, mileage, model year, fuel type, and brand information.

## Key Insights
- Lower mileage tends to correlate with higher price
- Some brands preserve value better
- Recent model years concentrate in premium ranges
